# Dual-Deep-v2-final 开发记录

## 版本定位

基于 v1 最小核心实现，通过系统性参数调优确定最优配置。

**核心发现**：v2 相比 v1 只需 **1 处关键修改** — 将 `HEUR` 中的 `* AQ[node]` 改为 `* powf(AQ[node], β)`，配合参数调整即可获得 +4~5% 的下界提升。

之前 v2 设计的 6 个复杂策略（Rank-based / β退火 / Position-aware / Pheromone Smoothing / Novelty / Entropy q₀）经消融实验证明：**复杂策略的额外开销抵消了理论收益**，简洁方案反而最优。

## 参数调优过程

### β 指数（最关键参数）

| β | T1_200 avg | T1_500 avg | T1_1000 avg |
|---|-----------|-----------|------------|
| 1.0 (v1) | 230.3 | 1584.3 | 2487.3 |
| 1.5 | 226.7 | 1622.7 | 2559.7 |
| 2.0 | 230.7 | 1630.0 | 2569.0 |
| 2.5 | 236.0 | 1624.3 | 2588.3 |
| **3.0** | **239.7** | **1640.0** | **2604.3** |
| 4.0 | 240.7 | 1631.0 | 2617.7 |

β=3.0 为最佳平衡点。

### ρ 蒸发率

| ρ | T1_200 | T1_500 | T1_1000 |
|---|--------|--------|---------|
| **0.05** | **244** | **1669** | **2613** |
| 0.10 | 238 | 1632 | 2592 |
| 0.20 | 237 | 1592 | 2540 |
| 0.30 | 226 | 1576 | 2487 |

ρ=0.05 完胜（β 放大 AQ 差异，需低蒸发率保持稳定）。

### q₀ 利用概率

q₀ ∈ {0.7, 0.8, 0.85, 0.9, 0.95} 差异小于 2%，取 q₀=0.9。

### K 蚂蚁数

K ∈ {3, 5, 7, 10} 差异小于 1%，保持 K=5。

## 最终参数

| 参数 | v1 值 | v2-final 值 | 说明 |
|------|-------|-------------|------|
| β | 1.0 (隐式) | **3.0** | AQ 影响力指数 |
| ρ | 0.1 | **0.05** | 蒸发率（降低以配合高 β） |
| q₀ | 0.95 | **0.9** | 利用概率（略降以维持探索） |
| K | 5 | 5 | 蚂蚁数（不变） |
| λ | 0.3 | 0.3 | Violated 惩罚系数（不变） |
| τ₀ | 1.0 | 1.0 | AQ 初始值（不变） |
| τ_min/max | 0.01/20 | 0.01/20 | AQ 范围（不变） |

## 5-seed 验证结果

| 配置 | T1_200 | T1_500 | T1_1000 | UDG_500 | UDG_1000 |
|------|--------|--------|---------|---------|----------|
| v1 | 228.6 | 1586.6 | 2489.6 | 832.0 | 100.2 |
| **v2-final** | **239.6** | **1649.2** | **2618.4** | **831.4** | **102.4** |
| 提升 | +4.8% | +3.9% | +5.2% | -0.07% | +2.2% |

v2 最差情况 > v1 最好情况（T1_1000: v2 min 2606 > v1 max 2521）。

## 代码修改清单（相对于 v1）

1. `lb-search-util.h` 第 197 行：加 `AQ_BETA = 3.0f` 参数
2. `lb-search-util.h` 第 195 行：`AQ_RHO` 默认值 0.1 → 0.05
3. `lb-search-util.h` 第 196 行：`AQ_Q0` 默认值 0.95 → 0.9 (可选)
4. `lb-search-util.h` HEUR 宏：`*AQ[node]` → `*powf(AQ[node],AQ_BETA)`
5. `main.cpp`：新增第 8 个命令行参数 `[beta]`

## 运行方式

```bash
./dual-deep-v2-bin <instance> <cutoff> <seed> <alpha> [K] [rho] [q0] [beta]
# 默认: K=5 rho=0.05 q0=0.9 beta=3.0
./dual-deep-v2-bin instance.wclq 10 1 90 5 0.05 0.9 3.0
```
