原始代码的数据流
main.cpp: while 循环（DBS rounds）
  └── ibmwds_init_bounds(round_k)         // 每个 round 调用一次
        ├── (round_k > 0) Beta[v] *= ALPHA   // γ 增长
        └── compute_bounds(round_k, cutoff)    // IT-LBS 核心
              │
              do-while 循环（迭代，最多 MaxIteration 次无改善）：
              │
              ├── [重置] memcpy DeltaCost/Node_Delta/MinNeibor，清 status
              ├── [建堆] build_node_domcost_heap_comb()
              ├── [构造排序] while 循环：堆顶取 HEUR 最高顶点
              │     └── 每步：取顶点 → 记录 PairList → 更新 delta
              ├── [评估] lb = Σdelta，更新 BEST_LOWER_BOUND
              ├── [副作用] compute_loss_and_upper_bound()
              │     └── 计算 replaced（violated）→ 更新 Loss/Subs → 计算 UB
              └── [终止] 检查 OPTIMAL / timeout

每只蚂蚁前后的状态变化图
蚂蚁 k 开始前：
  DeltaCost[] = DomCost[]       ← 从原始值恢复（memcpy）
  Node_Delta[] = Node_Weight[]  ← 从原始值恢复
  MinNeibor[] = MinNeiborBak[]  ← 从原始值恢复
  domed(v) = 0, selected(v) = 0 ← 全部清除

蚂蚁 k 执行中：
  堆选择 → PairList[0..m-1] 被填充（replaced=0）
         → PairIndex[target] 被设置
         → selected(target) 被设置
         → domed(相关节点) 被设置
         → DeltaCost/Node_Delta 被修改

蚂蚁 k 完成后：
  → 计算 lb_k = Σ PairList[i].delta
  → 调用 aq_count_violated() 得到 n_violated_k
    （注意：此函数利用当前的 selected() 和 PairIndex[]，不修改 Loss/Subs）
  → 计算 Score_k
  → 如果是最优蚁：memcpy PairList 到 BestAnt_PairList

下一只蚂蚁开始前：
  → 恢复 DeltaCost/Node_Delta/MinNeibor（memcpy）
  → 清除 domed/selected 状态
  → PairList 将被覆盖（不需要显式清除）
